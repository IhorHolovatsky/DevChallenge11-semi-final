Як запустити:

1. В папці Vagrant виконати команду "vangrant up", почекати поки запуститься віртуальна машина
2. Готово, WebApi буде доступна по цьому урл - http://127.0.0.1:1025/

Наприклад, найти документ за номером: http://127.0.0.1:1025/Document/GetDocumentByIdentifier/?documentIdentifier=479-17-06

Наприклад, синхронізувати документи: http://localhost:1025/Admin/SyncDocuments?publishStartDate=7/1/2017&publishEndDate=6/1/2017&token=E651A2A0-C5DC-4231-B8E6-B1D4F37CF0FF

 P.S: E651A2A0-C5DC-4231-B8E6-B1D4F37CF0FF - адмін токен

NOTES:

 Щоб не тратити багато часу на підняття MSSQL через vagrant у віртуальну машину, я направив всюди ConnString на базу в амазоні. Оскільки вона фізично знаходиться далеко (пінг ~600 мс), то перформанс не сильно високий.
 Можна підставити ваш локальний MSSQL сервер(достатньо лише поміняти Data Source в конн стрінгу).
 Використав Entity Framework Code First підхід, тому базу воно саме згенерує.
